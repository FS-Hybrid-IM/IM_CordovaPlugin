  • text-shadow: X轴 Y轴 Rpx color
  • 属性说明（顺序依次对应）
  • 阴影的X轴(可以使用负值) 阴影的Y轴(可以使用负值) 阴影模糊值（大小） 阴影的颜色
  • 此属性使用于文字阴影，而不是对盒模型进行操作 如果设置盒模型阴影请参考知识点:box-shadow（同理）
  • IE下使用滤镜filter:shadow()
  • 和box-shadow一样都是css3新增的属性，为了兼容各主流浏览器并支持这些主浏览器的较低版本，基于主流浏览器上使用text-shadow属性时，我们需要将属性的名称前面加上前缀
  • 朋浏览器 -o-text-shadow
  • -webkit-text-shadow:0 0 10px #c06;
  • -moz-text-shadow:0 0 10px #C06;
  • -o-text-shadow:0 0 10px #C06;
